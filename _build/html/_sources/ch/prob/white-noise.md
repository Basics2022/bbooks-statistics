# White noise


```{prf:definition} White noise - properties
:label: wn:def:properties

A white noise is a random process with

- zero expected value 

  $$\mathbb{E}[ \xi(t) ] = 0$$

- Dirac delta correlation

  $$\mathbb{E}[ \xi(t) \xi(s) ] = \delta(t-s)$$


```
